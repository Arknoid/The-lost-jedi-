Version de base :
->Scrolling vertical et horizontal, top-down.
->3 vies de départ + crédits
->Scoring
->Packs de mobs aléatoires avec timer
->Power-ups(speed up , speed, funnels, rayon chargé)
->Items (bombes thermiques, raid du faucon millenium)
->3 stages + boss rush
->3 types de mobs + boss (avec patterns)
->Drones, Tie-fighter, Boba Fett, Dark Vador, etc…
->Vaisseaux différents : X-wing, Snow Speeder
->Jauge de bouclier réflecteur
-> Store pour acheter améliorations en dépensant le score
-> System procédural par round
